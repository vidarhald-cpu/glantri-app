# @glantri/shared

Felleskode uten domeneavhengigheter: utilities, hjelpefunksjoner og typer som brukes på tvers av pakker.

Importerer ikke fra andre `@glantri/*`-pakker.

## Innhold

Typisk innhold: generelle TypeScript-utilities, pagination-typer, date-helpers og lignende.

## Importregel

Kan importeres av alle lag. Importerer ingenting fra `@glantri/*`.
